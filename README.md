# Accommodation-reservation
스프링 메이븐 프로젝트 + JSP 숙박예약 사이트

## 💡 프로젝트 소개
전국 숙박 업체 예약 검색 및 결제 사이트 입니다.<br>
<br>

#### 진행 기간
* 21.4.16 - 21.5.21

### ⚙️ 개발 환경
- `Java 8`
- `JDK 14.0.1`
- **IDE** : Eclipse
- **Server** : Apache Tomcat
- **Database** : Oracle DB
- **Framework** : Spring Framework
- **Library** : Mybatis

## 기능
1. 회원가입
   * 아이디, 비밀번호, 주소 API
   * 아이디 중복검사
   * ID찾기, PW찾기
2. 로그인
   * 일반 로그인, 관리자 로그인
   * 관리자 전용 탭
3. 회원 정보
   * 회원 정보 수정/탈퇴
4. 숙소 정보
   * 숙소 정보 수정/등록/삭제
   * 숙소 목록/객실 목록/등록
